Exercício de fixação de links internos

Exercício baseado nesta aula prática.

Serão criadas 5 páginas diferentes com o objetivo de treinar e fixar o assunto de links internos, para assim prosseguir para o próximo vídeo do curso, conforme combinado com o professor Gustavo Guanabara.

Vamos começar com o mais simples, vasmos para página 2, localizada nessa mesma pasta.

Será criado uma barra com os números das páginas para facilitar a navegação pelo site.
