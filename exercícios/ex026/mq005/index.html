Testando Media Queries

Exercício do capítulo 25 do curso de HTML e CSS do Curso em Vídeo, onde aprenderemos a criar um menu responsívo que se adapta a diversos dispositivos com tamanhos de telas diferentes.

Estamos fazendo este exercício seguindo a metodologia Mobile First, ou seja, começamos fazendo o design do site para aparelhos com telas pequenas e depois fazemos as configurações para as telas maiores. O intuito das aulas 9, 10 e 11 do capítulo 25 do curso é criar um menu responsívo para telas pequenas, portanto, focaremos na configuração do menu, que nas telas pequenas estarão um em cima do outro e por isso são chamados de "menu hambúrguer" pelo Guanabara.

Na aula 9 damos início ao projeto, criando a estrutura básica do HTML do site e algumas configurações básicas do CSS. Na aula 10 começamos a configurar o layout do menu, o deixando um em cima do outro. Na aula 11 começamos a criar a interatividade do menu hambúrguer usando o JavaScript, mas antes disso nós importamos uma folha de estilo do Google copiando um link do site Google Fonts e deste outro (este último inclusive ensina a como importar e usar esta folha de estilo), isso nos permite colocar o ícone do menu que você está vendo na página. Dentre esse, há vários outros ícones que podem ser usados.

E por fim, a interatividade. Nós usaremos o JavaScript para isso, primeiro escondemos as opções do menu com o display: none; no seletor do menu no CSS. Feito isso, nós criamos um id chamado "items" para a tag <menu> onde o ícone do menu está contido. Agora nós usaremos a tag <script> para escrever o código JavaScript.

Dentro da div span do ícone do menu, colocamos um evento onclick com uma function chamada "clickMenu". Dentro da tag <script>, escreveremos o seguinte código: 

                
            function clickMenu() {
                if (items.style.display == 'block') {
                    items.style.display = 'none'
                } else {
                    items.style.display = 'block'
                }
            }

A function clickMenu fará com que os items do menu, sejam mostrados (items.style.display == 'block'), porém, há uma condicional (if - else) para esconder-lo caso dele seja clicado novamente (items.style.display = 'none').
